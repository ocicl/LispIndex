## cl-coroutine

**System Name:** `cl-coroutine`

**Purpose:** Coroutine implementation built on cl-cont continuations.

**Why Use It:**
- **Cooperative multitasking**: Yield control between coroutines
- **Simple API**: Easy-to-use coroutine primitives
- **Continuation-based**: Built on solid cl-cont foundation

**Best For:** Asynchronous programming, generators, cooperative task scheduling, and state machines.
